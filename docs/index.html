Yet Another Audio To Video

Audio and cover art to video.

Turn a song, ASMR session, podcast episode, mix, or voice note into an upload-ready video file. Pick an audio file, pick a static cover image, and yaatv makes the video locally.

Download yaatv See examples 
yaatv -a audio.flac -i cover.jpg -o output.mp4 --resolution 1440p

What files work?

Use one finished audio file and one still image. yaatv is built for normal audio uploads, not video editing projects or animated artwork.

Audio

Common formats such as WAV, FLAC, MP3, M4A/AAC, OGG, and Opus.

Cover art

Static images such as JPG, PNG, and WebP. Animated images are rejected.

Result

An MP4 by default, or a ProRes MOV when the output path ends in .mov.

Download

Download the ZIP that matches your computer. On GitHub, choose one of these release assets and ignore the source code downloads unless you specifically want the code.

macOS x64 yaatv-macos-x64.zip yaatv-macos

yaatv is a command-line app. Extract the ZIP, open PowerShell or a terminal in that folder, then run the executable with --version. Run --install-ffmpeg once to install the FFmpeg tools yaatv uses. Interactive runs ask before installing FFmpeg. The macOS build is unsigned. Apple Silicon Macs may need Rosetta to run yaatv, and --install-ffmpeg supports Apple Silicon FFmpeg tools. Open the latest release.

Make a video

Most runs only need an audio path and an image path. If you do not choose an output name, yaatv uses the audio file name, or artist/title tags when they are available. 

yaatv -a audio.flac -i cover.jpg
yaatv -a episode.wav -i cover.png -o upload.mp4 --resolution 4k
yaatv -a mix.wav -i cover.jpg -o upload.mov
yaatv -a session.mp3 -i art.jpg --pad 2

What yaatv writes

The output shows the cover image for the full length of the audio and is checked after encoding.

  • MP4 output uses H.264 for broad playback compatibility and YouTube uploads.
  • MOV output uses ProRes 422 for workflows that need MOV files.
  • Audio is kept in an upload-friendly format.
  • Cover art keeps its aspect ratio. yaatv pads with black instead of stretching the image.
  • Finished files are checked before yaatv reports success.

FFmpeg

Release ZIPs do not bundle FFmpeg or FFprobe. On supported systems, yaatv can install the tools it needs with yaatv --install-ffmpeg. yaatv checks its app-managed bin directory before PATH. If you installed FFmpeg manually and added it to PATH, verify your setup with the commands below. 

ffmpeg -version
ffprobe -version
